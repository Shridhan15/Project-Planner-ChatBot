# Overview of ProjectPartner

ProjectPartner is a collaboration platform where students can post projects, join teams, discover recommended projects, and communicate through built-in messaging.

The platform focuses on:
- Connecting students with similar skills.
- Helping new developers collaborate on real-world projects.
- Allowing project owners to build teams for their ideas.
- Providing a smooth way to communicate between collaborators.

Key features:
1. Post new projects.
2. Browse and search projects.
3. Send join requests to project creators.
4. Accept or reject join requests.
5. Chat through real-time messaging.
6. View recommended projects.
7. Edit and update personal profile.

All user-generated projects are public and visible to everyone, but **important actions require login** such as creating a project, sending a join request, sending messages, and accessing recommendations.
