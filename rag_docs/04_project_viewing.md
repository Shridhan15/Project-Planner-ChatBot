# Viewing Projects

All users (logged-in or not) can browse projects.

Project cards show:
- Project image
- Title
- Description
- Required Skills
- Tech Stack
- Author name (clickable to view profile)

Logged-in users will also see:
- "Request to Join" button

Non-logged-in users do NOT see the join request button.

---

## Project Detail View
Clicking on a project card shows:
- Full project description
- Skills required
- Tech used
- Author profile link
- Join button (logged-in users only)
