# Messaging System

Messaging allows users to communicate directly within the platform.

---

## Sending a Message
Only logged-in users can send messages.

Steps:
1. Click the author's name on any project card.
2. Visit the author's profile page.
3. Click **"Send Message"**.
4. The messaging page opens.

---

## Messaging Page Features
- Left sidebar shows all users you have chatted with.
- Right panel shows chat history with the selected user.
- A text input allows sending new messages.
- Messages appear instantly and are stored permanently.
