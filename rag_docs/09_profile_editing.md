# Profile Editing

Users can view and edit their profile from the Dashboard.

Profile fields include:
- Name
- Mobile number
- Email (not editable)
- Year of study
- Skills (used for recommendations)
- Tech stack (used for recommendations)
- Profile image

Changes are applied immediately and influence recommendation results.
