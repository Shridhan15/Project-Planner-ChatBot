# Join Request System

Join requests allow users to request membership in a project.

---

## Sending a Join Project Request
Logged-in users can click **"Request to Join"** on any project card.

After sending the request:
- A notification is sent to the project creator.
- The request remains pending until the creator responds.

---

## Receiving Join Project Requests (Project Authors Only)
Project authors receive notifications under the Bell icon.

Each notification includes:
- The name of the requester.
- The project they applied for.
- Options to Accept or Reject.

---

## Handling Project Requests
- **Accept** → The requester becomes part of the team.
- **Reject** → The requester is informed via a notification.

Creators can manage requests any time from the notifications dropdown.

---

## Limitations
- Users cannot send requests to closed projects.
- Users cannot request to join their own project.
