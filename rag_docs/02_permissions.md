# User Permissions & Access Rules

This document lists what users can and cannot do based on login status.

---

## Features available WITHOUT login
- View homepage.
- Browse public projects.
- Search projects by skill, tech stack, or keywords.
- View project cards and project details.
- View author profiles (limited view).

Cannot:
- Post any project.
- Send join requests.
- Send messages.
- View notifications.
- Edit profile.
- View recommended projects.
- Access dashboard.

---

## Features available ONLY AFTER login
- Post a new project.
- Send join requests.
- Send messages to any project author.
- Receive notifications for requests and responses.
- Accept or reject join requests for owned projects.
- Close projects.
- View personalized recommended projects.
- Edit profile information.
- Access dashboard with posted projects.

---

## Project Creator Permissions
Only the creator of a project can:
- Accept or reject join requests.
- Close the project.
- View full details of joining users.
- Manage project status.
