# Creating a New Project

Only logged-in users can post new projects.

To post a project:
1. Go to the **Post a New Project** page.
2. Fill in the following fields:
   - Project Title
   - Project Description
   - Required Skills (comma-separated)
   - Tech Stack (comma-separated)
3. Upload an optional project image.
4. Click **Submit Project**.

Once submitted:
- The project becomes visible on the homepage.
- Other users can now send join requests.
- The project appears under the creator’s Dashboard → My Projects section.

Notes:
- Project details cannot be edited after submission.
- If changes are needed, the project must be deleted and recreated.
